This is simple photo filter for editing photo. Based with Python, Tkinter, OpenCV, and Pillow.
#  Features
  -adjust brightness
  -adjust contrast
  -adjust shadows
  -adjust exposure
  -adjust warmth
  -upload images (JPG, PNG)
  -save photo
  -open new photo
  -apply all changes
  -discard all changes
  
# How to run :
  1.Clone the repository
  2.Install dependencies:
  
      pip install opencv-python numpy pillow
     
  # Technologies
- Python 3.8+
- Tkinter — GUI
- OpenCV — image processing
- Pillow — image conversion
- NumPy — array operations
